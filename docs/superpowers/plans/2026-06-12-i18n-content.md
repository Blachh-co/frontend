# Localized Content Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Move localized UI copy into per-language JSON files and render Thai and Swedish content across all current localized routes instead of falling back to English.

**Architecture:** Replace the inline TypeScript dictionary with JSON-backed locale files in `messages/`, then fetch the selected locale once per localized route and pass typed dictionary slices into the page and shared UI components. Keep product data and commerce state unchanged while translating all static UI text rendered under `app/[lang]`.

**Tech Stack:** Next.js App Router, TypeScript, Node test runner, JSON locale dictionaries

---

- [ ] **Step 2: Update `lib/i18n.ts` to load the JSON files**

```ts
import en from "@/messages/en.json";
import th from "@/messages/th.json";
import sv from "@/messages/sv.json";

const dictionaries = { en, th, sv } as const;
export type Dictionary = (typeof dictionaries)[Locale];

export async function getDictionary(locale: Locale): Promise<Dictionary> {
  return dictionaries[locale];
}
```
